Specifications

  • Dimension: 2.165” x 1.468” x 0.603”
  • Weight: 1 oz
  • RF Transmission Frequency: 418 MHz
  • Battery Life: 1 year or more (2years in Basic non-supervised mode)*
  • Battery Voltage: 3.0V
    • *Function of activity

Disclaimer:

The myActive Alert™ does not detect falls with perfect accuracy. If a customer needs help, they must always push the help button on the wearable device or base unit.

Mytrex myActive Alert™ Features

  • Academically and clinically validated technology
  • Can be used with the Mytrex MXD and MXD3G units
  • Range of 600-1000 feet
  • Capable of sending separate fall and panic button signals
  • Auto-cancel feature (if steps are detected after fall alert is detected, signal will not be transmitted to base unit)
  • Monitored battery
  • Can operate as a supervised Personal Help Button (PHB)
  • Capable of sending unique alarm codes for fall-type events, button pushes, inactivity, supervisory alarms, and low battery alarms
  • Fall detection feature can be disabled via the button (locally) or the base unit (locally or remotely)
  • Button Only Mode
  • Basic Mode
  • Enhanced Mode
  • Inactivity alert (Enhanced Mode only)
  • Additional passive monitoring features (Enhanced Mode)
  • Made In The USA
